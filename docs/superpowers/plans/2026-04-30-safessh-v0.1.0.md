# safessh v0.1.0 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers-extended-cc:subagent-driven-development (recommended) or superpowers-extended-cc:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship safessh v0.1.0 — a personal CLI proxy that lets LLMs run gated SSH commands on user-configured projects, with full approval lifecycle (once/timed/always/deny/block), JSONL audit, and skill installation for Claude Code + AGENTS.md. Installable via Homebrew + curl.

**Architecture:** Rust workspace of seven crates with downward-only dependencies. CLI dispatches through policy → approvals → audit → SSH-driver subprocess → framed output. State on disk under XDG paths; TUI deferred to v0.2. OpenSSH binary handles the actual protocol; ControlMaster handles connection reuse.

**Tech Stack:** Rust stable, `clap` (CLI), `tokio` (async), `thiserror` (errors), `conch-parser` (shell AST), `sqlparser` (SQL AST), `tempfile`/`fs2` (atomic writes + locks), `keyring` (OS keychain), `serde` + `toml` (config), `regex` (redaction), `tracing` (structured logs), `cargo-dist` (releases), `assert_cmd` + `testcontainers` (E2E tests).

**Spec reference:** [`docs/superpowers/specs/2026-04-30-safessh-design.md`](../specs/2026-04-30-safessh-design.md). This plan covers §9 v0.1.0 only.

**Conventions:**
- Every task ends with a Conventional Commits-formatted commit.
- Every task that adds load-bearing security code includes a `// SAFETY-INVARIANT-N:` marker matching spec §7.2.
- Verify commands are exact; expected output is described per task.
- TDD: write failing test → minimal implementation → passing test → commit.

```rust
use thiserror::Error;

#[derive(Debug, Error)]
pub enum Error {
    #[error("config error: {0}")]
    Config(String),

    #[error("project not found: {0}")]
    ProjectNotFound(String),

    #[error("usage: {0}")]
    Usage(String),

    #[error("approval required: token={token}")]
    ApprovalRequired { token: String, categories: Vec<String> },

    #[error("blocked by rule {rule_id}: {reason}")]
    Blocked { rule_id: String, reason: String },

    #[error("denied: {0}")]
    Denied(String),

    #[error("yolo refused: disabled globally")]
    YoloRefused,

    #[error("ssh failure: {0}")]
    Ssh(String),

    #[error("connection failure: {0}")]
    Connection(String),

    #[error("output exceeded cap ({limit_bytes} bytes)")]
    OutputCapped { limit_bytes: u64 },

    #[error("storage error: {0}")]
    Storage(String),

    #[error("audit write failure: {0}")]
    AuditWriteFailed(String),

    #[error("io: {0}")]
    Io(#[from] std::io::Error),

    #[error("serde: {0}")]
    Serde(String),
}

impl Error {
    /// Map to the documented exit code from spec §7.1.
    pub fn exit_code(&self) -> i32 {
        match self {
            Error::Config(_) | Error::ProjectNotFound(_) => 1,
            Error::Usage(_) => 2,
            Error::ApprovalRequired { .. } => 10,
            Error::Blocked { .. } => 11,
            Error::Denied(_) => 12,
            Error::YoloRefused => 13,
            Error::Ssh(_) => 20,
            Error::Connection(_) => 21,
            Error::OutputCapped { .. } => 30,
            Error::Storage(_) | Error::Io(_) | Error::Serde(_) => 40,
            Error::AuditWriteFailed(_) => 50,
        }
    }

    pub fn error_class(&self) -> &'static str {
        match self {
            Error::Config(_) => "config",
            Error::ProjectNotFound(_) => "project_not_found",
            Error::Usage(_) => "usage",
            Error::ApprovalRequired { .. } => "approval_required",
            Error::Blocked { .. } => "blocked",
            Error::Denied(_) => "denied",
            Error::YoloRefused => "yolo_refused",
            Error::Ssh(_) => "ssh_failure",
            Error::Connection(_) => "connection_failure",
            Error::OutputCapped { .. } => "output_capped",
            Error::Storage(_) => "storage",
            Error::Io(_) => "io",
            Error::Serde(_) => "serde",
            Error::AuditWriteFailed(_) => "audit_write_failed",
        }
    }
}

pub type Result<T> = std::result::Result<T, Error>;
```

```rust
// crates/safessh-core/src/types.rs
use crate::error::{Error, Result};
use chrono::{DateTime, Utc};
use rand::Rng;
use serde::{Deserialize, Serialize};
use std::path::PathBuf;

#[derive(Clone, Debug, Eq, PartialEq, Hash, Serialize, Deserialize)]
pub struct ProjectId(String);

impl ProjectId {
    pub fn new(s: impl Into<String>) -> Result<Self> {
        let s = s.into();
        if s.is_empty()
            || s.contains('/')
            || s.contains(' ')
            || s.contains('\\')
            || s.starts_with('.')
        {
            return Err(Error::Usage(format!("invalid project id: {s:?}")));
        }
        Ok(Self(s))
    }

    pub fn as_str(&self) -> &str { &self.0 }
}

#[derive(Clone, Debug, Serialize, Deserialize)]
#[serde(untagged)]
pub enum Target {
    SshConfigAlias { name: String, ssh_config_alias: String },
    Inline {
        name: String,
        host: String,
        #[serde(default = "default_port")]
        port: u16,
        user: String,
        identity_file: Option<PathBuf>,
        proxy_jump: Option<String>,
        keychain_secret: Option<String>,
    },
}

fn default_port() -> u16 { 22 }

impl Target {
    pub fn name(&self) -> &str {
        match self {
            Target::SshConfigAlias { name, .. } | Target::Inline { name, .. } => name,
        }
    }
}

    pub binary: String,
    pub flags: Vec<String>,
    pub args: Vec<String>,

    pub pipes: Vec<ParsedCommand>,
    pub env_mutations: Vec<(String, String)>,

    Allow { matched_rule: Option<String>, source: AllowSource },
    RequireApproval { token: ApprovalToken, categories: Vec<String>, reason: String },
    Block { rule_id: String, pattern: String },
    Deny { reason: String },
}

#[derive(Clone, Debug, Serialize, Deserialize)]
pub enum AllowSource {
    DefaultPolicy,
    AlwaysRule(String),
    TimedRule { rule_id: String, expires_at: DateTime<Utc> },
    Yolo,
}

#[derive(Clone, Debug, Serialize, Deserialize, PartialEq, Eq, Hash)]
pub struct ApprovalToken(String);

impl ApprovalToken {
    pub fn generate() -> Self {
        const ALPHABET: &[u8] = b"abcdefghijkmnopqrstuvwxyz23456789"; // no 0/1/l/o for readability
        let mut rng = rand::thread_rng();
        let token: String = (0..6).map(|_| ALPHABET[rng.gen_range(0..ALPHABET.len())] as char).collect();
        Self(token)
    }

    pub fn as_str(&self) -> &str { &self.0 }
}

#[derive(Clone, Debug, Serialize, Deserialize)]
pub struct AuditEvent {
    pub schema_version: u32,
    pub timestamp: DateTime<Utc>,
    pub event_type: String,
    pub project: Option<String>,
    pub data: serde_json::Value,
    pub error_class: Option<String>,
    pub error_message: Option<String>,
}

impl AuditEvent {
    pub fn new(event_type: impl Into<String>) -> Self {
        Self {
            schema_version: 1,
            timestamp: Utc::now(),
            event_type: event_type.into(),
            project: None,
            data: serde_json::Value::Null,
            error_class: None,
            error_message: None,
        }
    }
}
```

```rust
// crates/safessh-core/src/redactor.rs
use regex::bytes::Regex;
use std::collections::HashMap;

#[derive(Clone, Copy, Debug, Eq, Hash, PartialEq)]
pub enum RedactionType {
    AwsAccessKey,
    AwsSecretKey,
    BearerToken,
    JwtToken,
    PrivateKeyBlock,
    PasswordParam,
    Custom(&'static str),
}

impl RedactionType {
    fn placeholder(&self) -> &'static [u8] {
        match self {
            RedactionType::AwsAccessKey => b"<REDACTED:aws_access_key>",
            RedactionType::AwsSecretKey => b"<REDACTED:aws_secret_key>",
            RedactionType::BearerToken => b"<REDACTED:bearer_token>",
            RedactionType::JwtToken => b"<REDACTED:jwt>",
            RedactionType::PrivateKeyBlock => b"<REDACTED:private_key_block>",
            RedactionType::PasswordParam => b"<REDACTED:password>",
            RedactionType::Custom(_) => b"<REDACTED:custom>",
        }
    }
}

pub struct Redactor {
    patterns: Vec<(RedactionType, Regex)>,
}

impl Default for Redactor {
    fn default() -> Self {
        let patterns = vec![
            (RedactionType::AwsAccessKey, Regex::new(r"AKIA[0-9A-Z]{16}").unwrap()),
            (RedactionType::AwsSecretKey, Regex::new(r"(?i)aws_secret_access_key\s*[=:]\s*[A-Za-z0-9/+=]{40}").unwrap()),
            (RedactionType::BearerToken, Regex::new(r"(?i)Bearer\s+[A-Za-z0-9._\-+/=]+").unwrap()),
            (RedactionType::JwtToken, Regex::new(r"eyJ[A-Za-z0-9_\-]+\.[A-Za-z0-9_\-]+\.[A-Za-z0-9_\-]+").unwrap()),
            (
                RedactionType::PrivateKeyBlock,
                Regex::new(r"(?s)-----BEGIN [A-Z ]+PRIVATE KEY-----.+?-----END [A-Z ]+PRIVATE KEY-----").unwrap(),
            ),
            (RedactionType::PasswordParam, Regex::new(r"(?i)password=[^\s&]+").unwrap()),
        ];
        Self { patterns }
    }
}

impl Redactor {
    pub fn with_pattern(mut self, ty: RedactionType, regex: Regex) -> Self {
        self.patterns.push((ty, regex));
        self
    }

    /// SAFETY-INVARIANT-6: This is the last byte transformation before output leaves
    /// safessh's process. Any caller that emits bytes to the LLM/user must pass them
    /// through `redact()` first.
    pub fn redact(&self, input: &[u8]) -> (Vec<u8>, HashMap<RedactionType, usize>) {
        let mut current = input.to_vec();
        let mut counts: HashMap<RedactionType, usize> = HashMap::new();
        for (ty, re) in &self.patterns {
            let placeholder = ty.placeholder();
            let mut count = 0usize;
            current = re
                .replace_all(&current, |_caps: &regex::bytes::Captures| {
                    count += 1;
                    placeholder.to_vec()
                })
                .into_owned();
            if count > 0 {
                counts.insert(*ty, count);
            }
        }
        (current, counts)
    }
}
```

```rust
pub fn write_string(path: &Path, contents: &str) -> std::io::Result<()> {
```

```rust
/// SAFETY-INVARIANT-12: Concurrent CLI/TUI invocations must not corrupt

pub struct LockedFile(File);

impl LockedFile {
    pub fn open_exclusive(path: &Path) -> std::io::Result<Self> {
        if let Some(parent) = path.parent() {
            std::fs::create_dir_all(parent)?;
        }
        let f = File::options().read(true).write(true).create(true).open(path)?;
        f.lock_exclusive()?;
        Ok(Self(f))
    }

    pub fn into_inner(self) -> File { self.0 }
}

impl Drop for LockedFile {
    fn drop(&mut self) {
        let _ = self.0.unlock();
    }
}
```

```rust
impl PendingStore {
    pub fn new(paths: &Paths) -> Self { Self { dir: paths.approvals_dir().join("pending") } }

    pub fn add(&self, request: &PendingRequest) -> Result<()> {
        let path = self.dir.join(format!("{}.toml", request.token));
        let toml = toml::to_string_pretty(request).map_err(|e| Error::Serde(e.to_string()))?;
        atomic::write_string(&path, &toml)?;
        Ok(())
    }

    pub fn take(&self, token: &ApprovalToken) -> Result<PendingRequest> {
        let path = self.dir.join(format!("{}.toml", token.as_str()));
        let raw = std::fs::read_to_string(&path)
            .map_err(|_| Error::Usage(format!("no pending approval: {}", token.as_str())))?;
        let req: PendingRequest = toml::from_str(&raw).map_err(|e| Error::Storage(e.to_string()))?;
        std::fs::remove_file(&path).ok();
        Ok(req)
    }

    pub fn cleanup_expired(&self, max_age_hours: i64) -> Result<usize> {
        if !self.dir.exists() { return Ok(0); }
        let cutoff = Utc::now() - Duration::hours(max_age_hours);
        let mut removed = 0;
        for entry in std::fs::read_dir(&self.dir)? {
            let entry = entry?;
            if let Ok(raw) = std::fs::read_to_string(entry.path()) {
                if let Ok(req) = toml::from_str::<PendingRequest>(&raw) {
                    if req.created_at < cutoff {
                        let _ = std::fs::remove_file(entry.path());
                        removed += 1;
                    }
                }
            }
        }
        Ok(removed)
    }
}

fn save_locked<T: Serialize>(path: &PathBuf, list: &RuleList<T>) -> Result<()> {
    let _lock = LockedFile::open_exclusive(path)?;
    let toml = toml::to_string_pretty(list).map_err(|e| Error::Serde(e.to_string()))?;
    atomic::write_string(path, &toml)?;
    Ok(())
}

fn load_or_default<T: for<'de> Deserialize<'de> + Default>(path: &PathBuf) -> Result<RuleList<T>>
where
    RuleList<T>: Default,
{
    if !path.exists() { return Ok(RuleList::default()); }
    let raw = std::fs::read_to_string(path)?;
    toml::from_str(&raw).map_err(|e| Error::Storage(e.to_string()))
}
```

```rust
// crates/safessh-policy/src/parser.rs
use crate::ast::ParsedCommand;
use thiserror::Error;

#[derive(Debug, Error)]
pub enum ParseError {
    #[error("opaque parse: {0}")]
    Opaque(String),
}

pub fn parse(raw: &str) -> Result<Vec<ParsedCommand>, ParseError> {
    use conch_parser::lexer::Lexer;
    use conch_parser::parse::DefaultParser;

    let lex = Lexer::new(raw.chars());
    let parser = DefaultParser::new(lex);
    let mut out = vec![];
    for cmd in parser {
        match cmd {
            Ok(top) => collect(&top, raw, &mut out)
                .map_err(|e| ParseError::Opaque(format!("ast walk: {e}")))?,
            Err(e) => return Err(ParseError::Opaque(format!("conch: {e:?}"))),
        }
    }
    if out.is_empty() {
        return Err(ParseError::Opaque("empty command".into()));
    }
    Ok(out)
}

fn collect(
    _top: &impl std::fmt::Debug,
    raw: &str,
    out: &mut Vec<ParsedCommand>,
) -> Result<(), String> {
    // Minimal first pass: split by simple tokenization. The AST walker
    // for full conch-parser nodes lands in step 4. For v0.1, we extract:
    //   binary, flags, args from the first simple command.
    // Pipes and redirects detection done via raw string scan in step 4.
    let trimmed = raw.trim();
    if trimmed.is_empty() { return Ok(()); }
    let parts: Vec<&str> = trimmed.split_whitespace().collect();
    if parts.is_empty() { return Ok(()); }
    let mut flags = vec![];
    let mut args = vec![];
    let binary = parts[0].to_string();
    for part in &parts[1..] {
        if part.starts_with("--") {
            flags.push(part.to_string());
        } else if part.starts_with('-') && part.len() > 1 {
            // Bundled short flags: -rf -> -r, -f
            for c in part[1..].chars() {
                flags.push(format!("-{c}"));
            }
        } else {
            args.push(part.to_string());
        }
    }
    out.push(ParsedCommand {
        binary,
        flags,
        args,
        redirects: vec![],
        pipes: vec![],
        env_mutations: vec![],
        raw: raw.to_string(),
    });
    Ok(())
}
```

```rust
    let mut out = vec![];

    if is_destructive_filesystem(cmd) { out.push("destructive:filesystem".into()); }
    if is_destructive_disk(cmd) { out.push("destructive:disk".into()); }
    if is_privilege_escalation(cmd) { out.push("privilege:escalation".into()); }
    if is_system_control(cmd) { out.push("system:control".into()); }
    if is_network_listen(cmd) { out.push("network:listen".into()); }
    if is_exec_opaque(cmd) { out.push("exec:opaque".into()); }

const READ_SAFE_BINS: &[&str] = &[
    "ls", "cat", "head", "tail", "grep", "stat", "file", "wc", "sort", "uniq",
    "which", "whereis", "pwd", "id", "whoami", "uname", "date", "uptime",
    "df", "du", "free", "ps", "top", "echo", "printf",
];

pub fn is_read_safe(cmd: &ParsedCommand) -> bool {
    if cmd.binary == "find" {
        // find is read-only only without -delete or -exec
        return !cmd.flags.iter().any(|f| f == "-delete")
            && !cmd.args.iter().any(|a| a == "-delete" || a == "-exec");
    }
    READ_SAFE_BINS.contains(&cmd.binary.as_str())
}

const DESTRUCTIVE_FS_BINS: &[&str] = &["rm", "rmdir", "unlink", "shred"];

pub fn is_destructive_filesystem(cmd: &ParsedCommand) -> bool {
    if DESTRUCTIVE_FS_BINS.contains(&cmd.binary.as_str()) { return true; }
    if cmd.binary == "find"
        && (cmd.flags.contains(&"-delete".to_string())
            || cmd.args.iter().any(|a| a == "-delete" || a == "-exec"))
    {
        return true;
    }
    false
}

const DESTRUCTIVE_DISK_BINS: &[&str] = &["dd", "fdisk", "parted", "wipefs", "mkfs"];

pub fn is_destructive_disk(cmd: &ParsedCommand) -> bool {
    if DESTRUCTIVE_DISK_BINS.contains(&cmd.binary.as_str()) { return true; }
    if cmd.binary.starts_with("mkfs.") { return true; }
    cmd.redirects.iter().any(|r| r.contains("/dev/sd")
        || r.contains("/dev/nvme")
        || r.contains("/dev/disk"))
}

const PRIV_ESC_BINS: &[&str] = &["sudo", "su", "doas", "pkexec"];

pub fn is_privilege_escalation(cmd: &ParsedCommand) -> bool {
    PRIV_ESC_BINS.contains(&cmd.binary.as_str())
}

const SYSTEM_CONTROL_BINS: &[&str] = &["shutdown", "reboot", "halt", "poweroff"];

pub fn is_system_control(cmd: &ParsedCommand) -> bool {
    if SYSTEM_CONTROL_BINS.contains(&cmd.binary.as_str()) { return true; }
    if cmd.binary == "systemctl" {
        return cmd.args.iter().any(|a|
            matches!(a.as_str(), "stop" | "disable" | "mask" | "kill"));
    }
    false
}

pub fn is_network_listen(cmd: &ParsedCommand) -> bool {
    if cmd.binary == "nc" || cmd.binary == "ncat" {
        return cmd.flags.contains(&"-l".to_string());
    }
    if cmd.binary == "socat" {
        return cmd.args.iter().any(|a| a.contains("LISTEN"));
    }
    if cmd.binary == "python" || cmd.binary == "python3" {
        return cmd.args.iter().any(|a| a == "http.server")
            && cmd.flags.iter().any(|f| f == "-m");
    }
    false
}

const KNOWN_SEMANTIC_INTERPRETERS: &[&str] = &["psql", "mysql", "sqlite3"];

pub fn is_exec_opaque(cmd: &ParsedCommand) -> bool {
    // Excludes known semantic interpreters; those route to db:* via Task 11.
    if KNOWN_SEMANTIC_INTERPRETERS.contains(&cmd.binary.as_str()) { return false; }
    let opaque_bins = ["sh", "bash", "zsh", "fish", "ksh", "dash"];
    if opaque_bins.contains(&cmd.binary.as_str())
        && cmd.flags.contains(&"-c".to_string())
    {
        return true;
    }
    if (cmd.binary == "python" || cmd.binary == "python3")
        && cmd.flags.contains(&"-c".to_string())
    {
        return true;
    }
    if (cmd.binary == "perl" || cmd.binary == "ruby")
        && cmd.flags.contains(&"-e".to_string())
    {
        return true;
    }
    if cmd.binary == "eval" { return true; }
    false
}
```

```rust
// crates/safessh-policy/src/categories/sql.rs
use safessh_core::types::ParsedCommand;
use sqlparser::ast::Statement;
use sqlparser::dialect::GenericDialect;
use sqlparser::parser::Parser;

pub fn match_sql_categories(cmd: &ParsedCommand) -> Vec<String> {
    let sql = match extract_sql(cmd) {
        Some(s) => s,
        None => return vec![],
    };
    let stmts = match Parser::parse_sql(&GenericDialect {}, &sql) {
        Ok(s) => s,
        Err(_) => return vec!["db:write".into()], // unknown SQL → conservative
    };
    let mut out = vec![];
    for stmt in stmts {
        match stmt {
            Statement::Query(_) => out.push("db:read".into()),
            Statement::Insert { .. } => out.push("db:write".into()),
            Statement::Update { .. } => out.push("db:write".into()),
            Statement::Delete(d) => {
                if d.selection.is_some() {
                    out.push("db:write".into());
                } else {
                    out.push("destructive:db".into());
                }
            }
            Statement::Drop { .. } => out.push("destructive:db".into()),
            Statement::Truncate { .. } => out.push("destructive:db".into()),
            Statement::CreateTable { .. } | Statement::CreateIndex { .. } => {
                out.push("db:write".into())
            }
            Statement::AlterTable { .. } => out.push("db:write".into()),
            Statement::Explain { .. } | Statement::ShowTables { .. }
            | Statement::ShowColumns { .. } => out.push("db:read".into()),
            _ => out.push("db:write".into()),
        }
    }
    out.sort();
    out.dedup();
    out
}

fn extract_sql(cmd: &ParsedCommand) -> Option<String> {
    match cmd.binary.as_str() {
        "psql" => extract_after_flag(cmd, "-c"),
        "mysql" => extract_after_flag(cmd, "-e"),
        "sqlite3" => cmd.args.iter().rfind(|a| !a.is_empty()).cloned(),
        _ => None,
    }
}

fn extract_after_flag(cmd: &ParsedCommand, flag: &str) -> Option<String> {
    // The minimal parser stores `-c` in flags; the SQL is in args.
    if cmd.flags.iter().any(|f| f == flag) {
        return cmd.args.first().cloned();
    }
    None
}
```

### Task 12: Decision engine

**Goal:** Pure function: `(parsed_command, project_policy, allow_store, timed_store, blocked_store) -> PolicyDecision`. Block-list checked first. Default-deny on parse failure.

**Files:**
- Create: `crates/safessh-policy/src/decision.rs`
- Modify: `crates/safessh-policy/src/lib.rs`
- Create: `crates/safessh-policy/tests/decision.rs`

**Acceptance Criteria:**
- [ ] If any block-rule matches the parsed command's pattern, returns `Block`.
- [ ] Else if project policy lists any matched category in `deny`, returns `Deny`.
- [ ] Else if any unexpired `timed` rule matches, returns `Allow { source: TimedRule }`.
- [ ] Else if any `always` rule matches, returns `Allow { source: AlwaysRule }`.
- [ ] Else if all matched categories are in project's `allow` list (and none in `require_approval` or `deny`), returns `Allow { source: DefaultPolicy }`.
- [ ] Else returns `RequireApproval` with token + categories + reason.
- [ ] Parse failure → `RequireApproval` with reason `"unparseable command"`.
- [ ] `// SAFETY-INVARIANT-1:` and `// SAFETY-INVARIANT-2:` markers.

**Verify:** `cargo test --package safessh-policy --test decision`

**Steps:**

- [ ] **Step 1: Implement `decision.rs`**

```rust
// crates/safessh-policy/src/decision.rs
use crate::categories::match_all;
use crate::parser::{parse, ParseError};
use safessh_core::types::{AllowSource, ApprovalToken, ParsedCommand, PolicyDecision};
use safessh_storage::approvals::{PatternRule, TimedRule};
use safessh_storage::project::Policy;

pub struct DecisionInput<'a> {
    pub raw: &'a str,
    pub policy: &'a Policy,
    pub allows: &'a [PatternRule],
    pub timed: &'a [TimedRule],
    pub blocks: &'a [PatternRule],
}

pub fn decide(input: DecisionInput) -> (PolicyDecision, Option<ParsedCommand>) {
    // SAFETY-INVARIANT-1: Parse failure produces RequireApproval, never Allow.
    let parsed = match parse(input.raw) {
        Ok(mut v) if !v.is_empty() => v.remove(0),
        _ => {
            return (
                PolicyDecision::RequireApproval {
                    token: ApprovalToken::generate(),
                    categories: vec!["unparseable".into()],
                    reason: "command could not be parsed safely".into(),
                },
                None,
            );
        }
    };
    let cats = match_all(&parsed);

    // SAFETY-INVARIANT-2: Block-list checked BEFORE allow-list.
    if let Some(rule) = matching_rule(&parsed, input.blocks) {
        return (
            PolicyDecision::Block {
                rule_id: rule.rule_id.clone(),
                pattern: format!("{} {}", rule.binary, rule.flags.join(" ")),
            },
            Some(parsed),
        );
    }

    if cats.iter().any(|c| input.policy.deny.contains(c)) {
        return (
            PolicyDecision::Deny {
                reason: format!("project denies categories: {cats:?}"),
            },
            Some(parsed),
        );
    }

    if let Some(rule) = matching_timed(&parsed, input.timed) {
        return (
            PolicyDecision::Allow {
                matched_rule: Some(rule.pattern.rule_id.clone()),
                source: AllowSource::TimedRule {
                    rule_id: rule.pattern.rule_id.clone(),
                    expires_at: rule.expires_at,
                },
            },
            Some(parsed),
        );
    }

    if let Some(rule) = matching_rule(&parsed, input.allows) {
        return (
            PolicyDecision::Allow {
                matched_rule: Some(rule.rule_id.clone()),
                source: AllowSource::AlwaysRule(rule.rule_id.clone()),
            },
            Some(parsed),
        );
    }

    let needs_approval = cats.iter().any(|c| input.policy.require_approval.contains(c))
        || cats.iter().any(|c| !input.policy.allow.contains(c));
    if !needs_approval {
        return (
            PolicyDecision::Allow {
                matched_rule: None,
                source: AllowSource::DefaultPolicy,
            },
            Some(parsed),
        );
    }

    (
        PolicyDecision::RequireApproval {
            token: ApprovalToken::generate(),
            categories: cats.clone(),
            reason: format!("requires approval for categories: {cats:?}"),
        },
        Some(parsed),
    )
}

fn matching_rule<'a>(parsed: &ParsedCommand, rules: &'a [PatternRule]) -> Option<&'a PatternRule> {
    rules.iter().find(|r| r.binary == parsed.binary && flags_match(&r.flags, &parsed.flags))
}

fn matching_timed<'a>(parsed: &ParsedCommand, rules: &'a [TimedRule]) -> Option<&'a TimedRule> {
    rules.iter().find(|r| {
        r.pattern.binary == parsed.binary && flags_match(&r.pattern.flags, &parsed.flags)
    })
}

fn flags_match(rule_flags: &[String], parsed_flags: &[String]) -> bool {
    rule_flags.iter().all(|f| parsed_flags.contains(f))
}
```

```rust
// crates/safessh-audit/src/jsonl.rs
use safessh_core::error::{Error, Result};
use safessh_core::redactor::Redactor;
use safessh_core::types::AuditEvent;
use safessh_storage::paths::Paths;
use std::fs::{File, OpenOptions};
use std::io::Write;
use std::path::PathBuf;

const ROTATE_AT: u64 = 100 * 1024 * 1024;

pub struct AuditWriter {
    path: PathBuf,
    redactor: Redactor,
}

impl AuditWriter {
    pub fn open(paths: &Paths) -> Result<Self> {
        if let Some(parent) = paths.audit_log().parent() {
            std::fs::create_dir_all(parent)?;
        }
        Ok(Self { path: paths.audit_log(), redactor: Redactor::default() })
    }

    /// SAFETY-INVARIANT-4: Audit append must succeed before any user-visible
    /// output. Returns Error::AuditWriteFailed (exit 50) if it cannot.
    pub fn append(&self, event: &AuditEvent) -> Result<()> {
        self.maybe_rotate()?;
        let json = serde_json::to_string(event)
            .map_err(|e| Error::AuditWriteFailed(format!("serialize: {e}")))?;
        let (redacted, _) = self.redactor.redact(json.as_bytes());
        let mut file = OpenOptions::new()
            .create(true).append(true)
            .open(&self.path)
            .map_err(|e| Error::AuditWriteFailed(format!("open: {e}")))?;
        #[cfg(unix)]
        {
            use std::os::unix::fs::PermissionsExt;
            let _ = std::fs::set_permissions(&self.path, std::fs::Permissions::from_mode(0o600));
        }
        file.write_all(&redacted).map_err(|e| Error::AuditWriteFailed(format!("write: {e}")))?;
        file.write_all(b"\n").map_err(|e| Error::AuditWriteFailed(format!("write nl: {e}")))?;
        file.sync_all().map_err(|e| Error::AuditWriteFailed(format!("fsync: {e}")))?;
        Ok(())
    }

    fn maybe_rotate(&self) -> Result<()> {
        if let Ok(meta) = std::fs::metadata(&self.path) {
            if meta.len() >= ROTATE_AT {
                let ts = chrono::Utc::now().format("%Y%m%dT%H%M%S");
                let rotated = self.path.with_file_name(format!("audit.log.{ts}"));
                let _ = std::fs::rename(&self.path, &rotated);
            }
        }
        Ok(())
    }
}
```

```rust
use crate::driver::{ExecResult, OutputChunk, SshDriver};
use async_trait::async_trait;

use safessh_storage::project::Target;
use std::collections::HashMap;
use std::sync::Mutex;

#[derive(Default)]
pub struct MockDriver {
    responses: Mutex<HashMap<(String, String), CannedResponse>>,
}

pub struct CannedResponse {
    pub stdout: Vec<u8>,
    pub stderr: Vec<u8>,
    pub exit: i32,
}

impl MockDriver {
    pub fn with_response(&self, target_name: &str, command: &str, r: CannedResponse) {
        self.responses.lock().unwrap().insert((target_name.into(), command.into()), r);
    }
}

#[async_trait]
impl SshDriver for MockDriver {
    async fn exec<'a>(
        &'a self,
        target: &'a Target,
        command: &'a str,
        _stdout_cap: u64,
        _stderr_cap: u64,
        mut on_chunk: Box<dyn FnMut(OutputChunk) + Send + 'a>,
    ) -> Result<ExecResult> {
        let key = (target.name().to_string(), command.to_string());
        let r = self.responses.lock().unwrap().remove(&key).unwrap_or(CannedResponse {
            stdout: vec![], stderr: vec![], exit: 0,
        });
        let stdout_len = r.stdout.len() as u64;
        let stderr_len = r.stderr.len() as u64;
        if !r.stdout.is_empty() { on_chunk(OutputChunk::Stdout(r.stdout)); }
        if !r.stderr.is_empty() { on_chunk(OutputChunk::Stderr(r.stderr)); }
        Ok(ExecResult {
            exit_code: r.exit,
            stdout_bytes: stdout_len, stderr_bytes: stderr_len,
            duration_ms: 0, truncated: false,
        })
    }
}
```

```rust
use crate::control_master;
use crate::driver::{ExecResult, OutputChunk, SshDriver};
use async_trait::async_trait;
use safessh_core::error::{Error, Result};
use safessh_storage::project::Target;
use std::path::PathBuf;
use std::time::Instant;
use tokio::io::AsyncReadExt;
use tokio::process::Command;

pub struct OpenSshDriver {
    control_dir: PathBuf,
}

impl OpenSshDriver {
    pub fn new(control_dir: PathBuf) -> Result<Self> {
        control_master::ensure_dir(&control_dir)?;
        Ok(Self { control_dir })
    }

    pub fn build_argv(&self, target: &Target, command: &str) -> Vec<String> {
        let mut argv = vec!["ssh".to_string()];
        argv.extend(control_master::argv_options(&self.control_dir));
        match target {
            Target::SshConfigAlias { ssh_config_alias, .. } => {
                argv.push(ssh_config_alias.clone());
            }
            Target::Inline { host, port, user, identity_file, proxy_jump, .. } => {
                argv.push("-p".into()); argv.push(port.to_string());
                if let Some(idf) = identity_file {
                    argv.push("-i".into()); argv.push(idf.display().to_string());
                }
                if let Some(pj) = proxy_jump {
                    argv.push("-J".into()); argv.push(pj.clone());
                }
                argv.push(format!("{user}@{host}"));
            }
        }
        argv.push("--".into());
        argv.push(command.to_string());
        argv
    }
}

#[async_trait]
impl SshDriver for OpenSshDriver {
    async fn exec<'a>(
        &'a self,
        target: &'a Target,
        command: &'a str,
        stdout_cap: u64,
        stderr_cap: u64,
        mut on_chunk: Box<dyn FnMut(OutputChunk) + Send + 'a>,
    ) -> Result<ExecResult> {
        let argv = self.build_argv(target, command);
        let started = Instant::now();
        let mut child = Command::new(&argv[0])
            .args(&argv[1..])
            .stdout(std::process::Stdio::piped())
            .stderr(std::process::Stdio::piped())
            .stdin(std::process::Stdio::null())
            .spawn()
            .map_err(|e| Error::Ssh(format!("spawn: {e}")))?;

        let mut stdout = child.stdout.take().unwrap();
        let mut stderr = child.stderr.take().unwrap();
        let mut stdout_total = 0u64;
        let mut stderr_total = 0u64;
        let mut truncated = false;
        let mut buf = vec![0u8; 8192];

        loop {
            tokio::select! {
                n = stdout.read(&mut buf) => match n {
                    Ok(0) => break,
                    Ok(n) => {
                        let allowed = stdout_cap.saturating_sub(stdout_total);
                        let take = (n as u64).min(allowed) as usize;
                        if take > 0 { on_chunk(OutputChunk::Stdout(buf[..take].to_vec())); }
                        stdout_total += take as u64;
                        if (n as u64) > allowed { truncated = true; let _ = child.kill().await; break; }
                    }
                    Err(_) => break,
                },
                n = stderr.read(&mut buf) => match n {
                    Ok(0) => continue,
                    Ok(n) => {
                        let allowed = stderr_cap.saturating_sub(stderr_total);
                        let take = (n as u64).min(allowed) as usize;
                        if take > 0 { on_chunk(OutputChunk::Stderr(buf[..take].to_vec())); }
                        stderr_total += take as u64;
                        if (n as u64) > allowed { truncated = true; let _ = child.kill().await; break; }
                    }
                    Err(_) => continue,
                },
            }
        }

        let status = child.wait().await.map_err(|e| Error::Ssh(format!("wait: {e}")))?;
        Ok(ExecResult {
            exit_code: status.code().unwrap_or(-1),
            stdout_bytes: stdout_total,
            stderr_bytes: stderr_total,
            duration_ms: started.elapsed().as_millis() as u64,
            truncated,
        })
    }
}
```

```rust
use crate::adapters::{filename, format, Target};
use crate::CONTENT;
use safessh_core::error::{Error, Result};
use safessh_storage::atomic;
use std::path::{Path, PathBuf};

#[derive(Clone, Copy, Debug)]
pub enum Scope { User, Project, Path }

pub fn install_to(target: Target, dest: &Path) -> Result<()> {
    let body = format(target, CONTENT);
    if matches!(target, Target::AgentsMd) {
        return install_agents_md_section(dest, &body);
    }
    atomic::write_string(dest, &body).map_err(Error::Io)?;
    Ok(())
}

fn install_agents_md_section(path: &Path, body: &str) -> Result<()> {
    let existing = std::fs::read_to_string(path).unwrap_or_default();
    let cleaned = strip_safessh_section(&existing);
    let combined = if cleaned.trim().is_empty() {
        body.to_string()
    } else {
        format!("{}\n\n{}", cleaned.trim_end(), body)
    };
    atomic::write_string(path, &combined).map_err(Error::Io)?;
    Ok(())
}

fn strip_safessh_section(content: &str) -> String {
    // Remove the existing ## safessh section if present.
    let lines: Vec<&str> = content.lines().collect();
    let mut out = vec![];
    let mut skipping = false;
    for line in &lines {
        if line.trim_start().starts_with("## safessh") {
            skipping = true;
            continue;
        }
        if skipping {
            // stop skipping at next top-level header
            if line.starts_with("## ") || line.starts_with("# ") {
                skipping = false;
                out.push(*line);
            }
            continue;
        }
        out.push(*line);
    }
    out.join("\n")
}

pub fn uninstall_at(target: Target, dest: &Path) -> Result<()> {
    if matches!(target, Target::AgentsMd) {
        let existing = std::fs::read_to_string(dest).unwrap_or_default();
        let cleaned = strip_safessh_section(&existing);
        atomic::write_string(dest, cleaned.trim_end()).map_err(Error::Io)?;
        return Ok(());
    }
    if dest.exists() {
        std::fs::remove_file(dest).map_err(Error::Io)?;
    }
    Ok(())
}

pub fn default_path(target: Target, scope: Scope, cwd: &Path) -> Option<PathBuf> {
    let home = dirs::home_dir()?;
    Some(match (target, scope) {
        (Target::ClaudeCode, Scope::User) => home.join(".claude/skills/safessh.md"),
        (Target::ClaudeCode, Scope::Project) => cwd.join(".claude/skills/safessh.md"),
        (Target::AgentsMd, Scope::Project) => cwd.join("AGENTS.md"),
        _ => return None,
    })
}

pub fn current_hash() -> String {
    use std::collections::hash_map::DefaultHasher;
    use std::hash::{Hash, Hasher};
    let mut h = DefaultHasher::new();
    CONTENT.hash(&mut h);
    format!("{:x}", h.finish())
```

```rust
// crates/safessh-cli/src/commands/exec.rs
use crate::output;
use safessh_audit::event;
use safessh_audit::jsonl::AuditWriter;
use safessh_core::error::{Error, Result};
use safessh_core::redactor::Redactor;
use safessh_core::types::PolicyDecision;
use safessh_policy::{decide, DecisionInput};
use safessh_ssh::driver::{OutputChunk, SshDriver};
use safessh_ssh::openssh::OpenSshDriver;
use safessh_storage::approvals::{AlwaysStore, BlockedStore, PendingRequest, PendingStore, TimedStore};
use safessh_storage::paths::Paths;
use safessh_storage::project::ProjectStore;
use std::sync::{Arc, Mutex};

pub async fn run(args: Vec<String>) -> Result<()> {
    if args.len() < 3 || args[1] != "exec" {
        return Err(Error::Usage(
            "expected: safessh <project> exec \"<command>\"".into()));
    }
    let project_name = &args[0];
    let raw_command = &args[2];

    let paths = Paths::user().map_err(Error::Io)?;
    paths.ensure_dirs().map_err(Error::Io)?;
    let project = ProjectStore::new(Paths::user().map_err(Error::Io)?).load(project_name)?;

    // Reload paths after move (or just re-derive). Simpler: derive fresh.
    let paths = Paths::user().map_err(Error::Io)?;
    let pending = PendingStore::new(&paths);
    let timed = TimedStore::new(&paths);
    let always = AlwaysStore::new(&paths);
    let blocked = BlockedStore::new(&paths);

    // Purge expired before decision so list_active is accurate.
    let _ = timed.purge_expired(project_name);

    let timed_rules = timed.list_active(project_name).unwrap_or_default();
    let allow_rules = always.list(project_name).unwrap_or_default();
    let block_rules = blocked.list(project_name).unwrap_or_default();

    let (decision, parsed) = decide(DecisionInput {
        raw: raw_command,
        policy: &project.policy,
        allows: &allow_rules,
        timed: &timed_rules,
        blocks: &block_rules,
    });

    let writer = AuditWriter::open(&paths)?;

    let parsed = parsed.unwrap_or_else(|| safessh_core::types::ParsedCommand {
        binary: "<unparsed>".into(), flags: vec![], args: vec![],
        redirects: vec![], pipes: vec![], env_mutations: vec![],
        raw: raw_command.clone(),
    });

    match &decision {
        PolicyDecision::Allow { source, .. } => {
            writer.append(&event::exec_attempt(project_name, &parsed, &format!("{source:?}")))?;
        }
        PolicyDecision::RequireApproval { token, categories, reason } => {
            let req = PendingRequest {
                token: token.as_str().to_string(),
                project: project_name.clone(),
                categories: categories.clone(),
                parsed: parsed.clone(),
                raw: raw_command.clone(),
                created_at: chrono::Utc::now(),
            };
            pending.add(&req)?;
            writer.append(&event::approval_requested(project_name, token.as_str(), categories, raw_command))?;
            return Err(Error::ApprovalRequired {
                token: token.as_str().to_string(),
                categories: categories.clone(),
            });
        }
        PolicyDecision::Block { rule_id, pattern } => {
            return Err(Error::Blocked { rule_id: rule_id.clone(), reason: pattern.clone() });
        }
        PolicyDecision::Deny { reason } => {
            return Err(Error::Denied(reason.clone()));
        }
    }

    // Allowed. Execute.
    let target = project.targets.iter()
        .find(|t| t.name() == project.default_target)
        .ok_or_else(|| Error::Config(format!("no target named {}", project.default_target)))?;

    let driver = OpenSshDriver::new(paths.cache.join("control-sockets")).map_err(|e| Error::Ssh(e.to_string()))?;
    let stdout_buf = Arc::new(Mutex::new(Vec::<u8>::new()));
    let stderr_buf = Arc::new(Mutex::new(Vec::<u8>::new()));
    let so = stdout_buf.clone();
    let se = stderr_buf.clone();
    let result = driver.exec(target, raw_command,
        project.output.stdout_cap_bytes, project.output.stderr_cap_bytes,
        Box::new(move |c| match c {
            OutputChunk::Stdout(b) => so.lock().unwrap().extend_from_slice(&b),
            OutputChunk::Stderr(b) => se.lock().unwrap().extend_from_slice(&b),
        })).await?;

    let r = Redactor::default();
    let stdout_red = r.redact(&stdout_buf.lock().unwrap()).0;
    let stderr_red = r.redact(&stderr_buf.lock().unwrap()).0;
    output::write_framed(&stdout_red, &stderr_red, result.exit_code, result.duration_ms, result.truncated);

    writer.append(&event::exec_complete(project_name, result.exit_code,
        result.stdout_bytes, result.stderr_bytes, result.duration_ms))?;

    if result.exit_code != 0 { std::process::exit(result.exit_code); }
    Ok(())
}
```

```rust
use safessh_core::error::Result;
use safessh_core::types::ParsedCommand;

#[derive(Debug, Clone)]
pub enum PromptAction { Once, Timed(u32), Always, Deny, Block }

pub fn ask(parsed: &ParsedCommand, categories: &[String], default_minutes: u32) -> Result<PromptAction> {
    if let Ok(canned) = std::env::var("SAFESSH_PROMPT_RESPONSE") {
        return Ok(parse_canned(&canned, default_minutes));
    }
    use dialoguer::Select;
    eprintln!("Command: {}", parsed.raw);
    eprintln!("Categories: {}", categories.join(", "));
    let items = &["once", "timed", "always", "deny", "block"];
    let sel = Select::new().with_prompt("Action").items(items).default(0).interact()
        .map_err(|e| safessh_core::error::Error::Usage(e.to_string()))?;
    Ok(match sel {
        0 => PromptAction::Once,
        1 => PromptAction::Timed(default_minutes),
        2 => PromptAction::Always,
        3 => PromptAction::Deny,
        _ => PromptAction::Block,
    })
}

fn parse_canned(s: &str, default_minutes: u32) -> PromptAction {
    if let Some(rest) = s.strip_prefix("timed:") {
        let m = rest.parse().unwrap_or(default_minutes);
        return PromptAction::Timed(m);
    }
    match s {
        "once" => PromptAction::Once,
        "timed" => PromptAction::Timed(default_minutes),
        "always" => PromptAction::Always,
        "deny" => PromptAction::Deny,
        "block" => PromptAction::Block,
        _ => PromptAction::Deny,
    }
}
```

```rust
        rule_id: format!("rule-{}", Utc::now().timestamp_millis()),
        binary: req.parsed.binary.clone(),
        flags: req.parsed.flags.clone(),
        args_pattern: None,
        categories: req.categories.clone(),
        created_at: Utc::now(),
    };

    if block {
        BlockedStore::new(&paths).add(&req.project, pattern)?;
        println!("Blocked persistently.");
    } else if always {
        AlwaysStore::new(&paths).add(&req.project, pattern)?;
        println!("Granted always.");
    } else if timed {
        let mins = minutes.unwrap_or(30);
        TimedStore::new(&paths).add(&req.project, TimedRule {
            pattern, expires_at: Utc::now() + Duration::minutes(mins as i64),
        })?;
        println!("Granted for {mins} minutes.");
    } else {
        println!("Granted once.");
    }
    Ok(())
}
```

```rust
#![cfg(feature = "integration")]

use assert_cmd::Command;
use predicates::str::contains;
use std::path::PathBuf;
use std::time::Instant;

fn fixtures_dir() -> PathBuf {
    PathBuf::from(env!("CARGO_MANIFEST_DIR"))
        .ancestors().nth(2).unwrap().join("tests/fixtures")
}

async fn start_ssh_container() -> (impl Drop, u16) {
    use testcontainers::runners::AsyncRunner;
    use testcontainers_modules::openssh::OpenSsh;
    let pubkey = std::fs::read_to_string(fixtures_dir().join("id_ed25519.pub")).unwrap();
    let img = OpenSsh::default().with_authorized_key(pubkey);
    let container = img.start().await.unwrap();
    let port = container.get_host_port_ipv4(22).await.unwrap();
    (container, port)
}

fn write_project(dir: &std::path::Path, port: u16) {
    let projects = dir.join("config/projects");
    std::fs::create_dir_all(&projects).unwrap();
    let id = fixtures_dir().join("id_ed25519");
    let toml = format!(r#"
name = "demo"
default_target = "default"

[[targets]]
name = "default"
host = "127.0.0.1"
port = {port}
user = "linuxserver.io"
identity_file = "{}"

[policy]
allow = ["read:safe"]
require_approval = ["destructive:filesystem"]

[approvals]
timed_default_minutes = 30
"#, id.display());
    std::fs::write(projects.join("demo.toml"), toml).unwrap();

#[tokio::test(flavor = "multi_thread")]
async fn happy_path() {
    let (_ct, port) = start_ssh_container().await;
    let home = tempfile::tempdir().unwrap();
    write_project(home.path(), port);
    Command::cargo_bin("safessh").unwrap()
        .env("SAFESSH_HOME", home.path())
        .args(["demo", "exec", "echo hello"])
        .assert().success()
        .stdout(contains("hello"))
        .stdout(contains("<exit code=\"0\""));
}

#[tokio::test(flavor = "multi_thread")]
async fn approval_round_trip() {
    let (_ct, port) = start_ssh_container().await;
    let home = tempfile::tempdir().unwrap();
    write_project(home.path(), port);
    let out = Command::cargo_bin("safessh").unwrap()
        .env("SAFESSH_HOME", home.path())
        .args(["demo", "exec", "rm -rf /tmp/x"])
        .output().unwrap();
    assert_eq!(out.status.code(), Some(10));
    let stderr = String::from_utf8_lossy(&out.stderr);
    let token = stderr.lines()
        .find(|l| l.starts_with("Token:"))
        .and_then(|l| l.split_whitespace().nth(1))
        .expect("token line in stderr");
    Command::cargo_bin("safessh").unwrap()
        .env("SAFESSH_HOME", home.path())
        .args(["approve", token])
        .assert().success();
    Command::cargo_bin("safessh").unwrap()
        .env("SAFESSH_HOME", home.path())
        .args(["demo", "exec", "rm -rf /tmp/x"])
        .assert().success();
}

#[tokio::test(flavor = "multi_thread")]
async fn control_master_reuse_is_faster() {
    let (_ct, port) = start_ssh_container().await;
    let home = tempfile::tempdir().unwrap();

    let cmd = || Command::cargo_bin("safessh").unwrap()

        .args(["demo", "exec", "echo hi"]).output().unwrap();

    let t1 = Instant::now();
    cmd();
    let d1 = t1.elapsed();
    let t2 = Instant::now();
    cmd();
    let d2 = t2.elapsed();
    assert!(d2 < d1, "second invocation must be faster (got {d1:?} vs {d2:?})");
}
```
