# safessh v0.2.0 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers-extended-cc:subagent-driven-development (recommended) or superpowers-extended-cc:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship safessh v0.2.0 — add the `safessh-tui` crate (four interactive screens for projects, approvals, rules, audit), enable real multi-target dispatch via `--on <target>`, import targets from the user's `~/.ssh/config`, keep TUI live via `notify` filesystem watching, and ship three new docs (`projects.md`, `tui.md`, `approvals.md`).

**Architecture:** A new `safessh-tui` crate (ratatui + crossterm) sits between the CLI and the existing storage/audit/policy crates — it consumes the same public APIs, never bypasses them, and writes through the same atomic + locked path everything else uses (SAFETY-INVARIANT-12). A single tokio task multiplexes terminal input and `notify` filesystem events into one `AppEvent` stream, so the screens don't poll. Multi-target lands in two parts: a global `--on <target>` flag on the CLI's exec path that resolves a named target out of `Project::targets`, and a `safessh project target add/list/remove` subcommand group. ssh-config import is a `safessh-storage::ssh_config` parser shared by both the CLI (`project add --import-ssh-config <alias>`) and the TUI projects screen.

**Tech Stack:** Rust stable, `ratatui` 0.28 (TUI rendering), `crossterm` 0.28 (terminal backend), `notify` 6 + `notify-debouncer-mini` (filesystem watching), `ssh2-config` 0.3 (ssh_config parser), existing `tokio`/`serde`/`toml`/`thiserror` from v0.1, `insta` snapshots over `ratatui::TestBackend` for screen tests.

**Spec reference:** [`docs/superpowers/specs/2026-04-30-safessh-design.md`](../specs/2026-04-30-safessh-design.md). This plan covers §9 v0.2.0 (and the v0.2-deferred bits of §5.6, §5.8, §6.6, §4.4 multi-target schema). The rest of the spec is shipped or deferred to later milestones.

**Conventions:**
- Every task ends with a Conventional Commits message (`feat(tui): …`, `feat(cli): …`, `docs: …`, `chore: …`).
- Every task that touches policy, audit, or storage adds a `// SAFETY-INVARIANT-N: <one-line reason>` comment naming the invariant it preserves, plus a test that asserts it.

```rust
//! Parse `~/.ssh/config` host aliases for project import.
//!
//! Cached to `~/.cache/safessh/ssh-config-snapshot.toml` so the TUI projects
//! screen doesn't re-parse on every key press. mtime-invalidated.

use crate::atomic;
use crate::paths::Paths;
use safessh_core::error::{Error, Result};
use serde::{Deserialize, Serialize};
use std::path::PathBuf;
use std::time::SystemTime;

/// SSH-config alias snapshot.
///
/// `ProxyJump` is intentionally omitted: ssh2-config 0.3 routes that
/// directive into its `unsupported_fields` sink rather than exposing a
/// typed field. Users who need ProxyJump should reference the alias via
/// `Target::SshConfigAlias` (`safessh project add --alias <name>`),
/// which delegates to `ssh` at exec time and respects the full ssh-config.
#[derive(Debug, Clone, Serialize, Deserialize, PartialEq, Eq)]
pub struct SshAlias {
    pub alias: String,
    pub hostname: Option<String>,
    pub user: Option<String>,
    pub port: Option<u16>,
    pub identity_file: Option<PathBuf>,
}

#[derive(Debug, Clone, Serialize, Deserialize, Default)]
pub struct SshConfigSnapshot {
    pub source_mtime_secs: i64,
    pub aliases: Vec<SshAlias>,
}

fn source_path() -> PathBuf {
    if let Ok(p) = std::env::var("SSH_CONFIG_PATH") {
        return PathBuf::from(p);
    }
    let home = std::env::var("HOME").unwrap_or_default();
    PathBuf::from(home).join(".ssh/config")
}

impl SshConfigSnapshot {
    pub fn load(paths: &Paths) -> Result<Self> {
        let src = source_path();
        let src_mtime = match std::fs::metadata(&src) {
            Ok(m) => m
                .modified()
                .ok()
                .and_then(|t| t.duration_since(SystemTime::UNIX_EPOCH).ok())
                .map(|d| d.as_secs() as i64)
                .unwrap_or(0),
            Err(_) => return Ok(Self::default()),
        };

        let cache_path = paths.ssh_config_snapshot();
        if let Ok(raw) = std::fs::read_to_string(&cache_path) {
            if let Ok(cached) = toml::from_str::<Self>(&raw) {
                if cached.source_mtime_secs == src_mtime {
                    return Ok(cached);
                }
            }
        }

        let aliases = parse_file(&src)?;
        let snap = Self {
            source_mtime_secs: src_mtime,
            aliases,
        };
        // SAFETY-INVARIANT-5: tempfile + rename via atomic::write_string.
        let toml = toml::to_string_pretty(&snap).map_err(|e| Error::Serde(e.to_string()))?;
        atomic::write_string(&cache_path, &toml)?;
        Ok(snap)
    }
}

fn parse_file(path: &std::path::Path) -> Result<Vec<SshAlias>> {
    use ssh2_config::{ParseRule, SshConfig};
    let raw = std::fs::read_to_string(path).map_err(Error::Io)?;
    let cfg = SshConfig::default()
        .parse(&mut raw.as_bytes(), ParseRule::ALLOW_UNKNOWN_FIELDS)
        .map_err(|e| Error::Storage(format!("ssh-config parse: {e}")))?;

    let mut out = Vec::new();
    // ssh2-config 0.3 exposes hosts as `cfg.get_hosts() -> &Vec<Host>`
    // where each `Host { pattern: Vec<HostClause>, params: HostParams }`.
    // One concrete clause = one alias; a `Host foo bar` block emits two.
    for host in cfg.get_hosts() {
        for clause in &host.pattern {
            if clause.negated { continue; }
            let alias = clause.pattern.clone();
            // Skip wildcard / glob entries — useless for import.
            if alias.contains('*') || alias.contains('?') || alias.is_empty() {
                continue;
            }
            out.push(SshAlias {
                alias,
                hostname: host.params.host_name.clone(),
                user: host.params.user.clone(),
                port: host.params.port,
                identity_file: host.params.identity_file.as_ref()
                    .and_then(|v| v.first().cloned()),
            });
        }
    }
    out.sort_by(|a, b| a.alias.cmp(&b.alias));
    // First-seen wins for duplicate Host stanzas.
    out.dedup_by_key(|a| a.alias.clone());
    Ok(out)
}
```

```rust
fn target_run(store: &ProjectStore, cmd: TargetCmd) -> Result<()> {
    match cmd {
        TargetCmd::Add { project, name, alias, host, user, port, identity, proxy_jump } => {
            let mut p = store.load(&project)?;
            if p.targets.iter().any(|t| t.name() == name) {
                return Err(Error::Usage(format!("target already exists: {name}")));
            }
            let target = match (alias, host, user) {
                (Some(a), _, _) => Target::SshConfigAlias { name: name.clone(), ssh_config_alias: a },
                (None, Some(h), Some(u)) => Target::Inline {
                    name: name.clone(), host: h, port, user: u,
                    identity_file: identity, proxy_jump, keychain_secret: None,
                },
                _ => return Err(Error::Usage("specify --alias OR (--host AND --user)".into())),
            };
            p.targets.push(target);
            store.save(&p)?;
            println!("Added target: {name}");
        }
        TargetCmd::List { project } => {
            let p = store.load(&project)?;
            for t in &p.targets {
                let marker = if t.name() == p.default_target { " [default]" } else { "" };
                let detail = match t {
                    Target::SshConfigAlias { ssh_config_alias, .. } => format!("alias={ssh_config_alias}"),
                    Target::Inline { host, port, user, .. } => format!("{user}@{host}:{port}"),
                };
                println!("{}{}  {}", t.name(), marker, detail);
            }
        }
        TargetCmd::Remove { project, name } => {
            let mut p = store.load(&project)?;
            if p.default_target == name {
                return Err(Error::Config("cannot remove default target".into()));
            }
            let before = p.targets.len();
            p.targets.retain(|t| t.name() != name);
            if p.targets.len() == before {
                return Err(Error::Config(format!("no such target: {name}")));
            }
            store.save(&p)?;
            println!("Removed target: {name}");
        }
    }
    Ok(())
```

```rust
        ]).split(frame.area());

struct Tui { terminal: Terminal<CrosstermBackend<Stdout>> }
impl Tui {
    fn enter() -> Result<Self> {
        enable_raw_mode().map_err(safessh_core::error::Error::Io)?;
        let mut stdout = std::io::stdout();
        execute!(stdout, EnterAlternateScreen, EnableMouseCapture)
            .map_err(safessh_core::error::Error::Io)?;
        let backend = CrosstermBackend::new(stdout);
        let terminal = Terminal::new(backend).map_err(safessh_core::error::Error::Io)?;
        Ok(Self { terminal })
    }
}
impl Drop for Tui {
    fn drop(&mut self) {
        let _ = disable_raw_mode();
        let _ = execute!(std::io::stdout(), LeaveAlternateScreen, DisableMouseCapture);
    }
}

pub async fn run(paths: Paths) -> Result<()> {
    let mut tui = Tui::enter()?;
    let mut app = App::new(paths);
    let mut events = EventStream::new()?;

    loop {
        tui.terminal.draw(|f| app.render(f)).map_err(safessh_core::error::Error::Io)?;
        match events.next().await {
            Some(AppEvent::Key(k)) => match app.handle_key(k) {
                AppAction::Quit => break,
                _ => {}
            },
            Some(AppEvent::Tick) => {}
            Some(AppEvent::Fs(_)) => {} // wired in Task 6
            None => break,
        }
    }
    Ok(())
}
```

```rust
use crossterm::event::{Event as CtEvent, KeyEvent};
use safessh_core::error::{Error, Result};
use std::time::Duration;
use tokio::sync::mpsc;

#[derive(Debug, Clone)]
pub enum FsEvent { ApprovalsChanged, ProjectsChanged, AuditAppended }

#[derive(Debug, Clone)]
pub enum AppEvent { Key(KeyEvent), Tick, Fs(FsEvent) }

pub struct EventStream {
    rx: mpsc::Receiver<AppEvent>,
    pub fs_tx: mpsc::Sender<FsEvent>,
}

impl EventStream {
    pub fn new() -> Result<Self> {
        let (tx, rx) = mpsc::channel(64);
        let (fs_tx, mut fs_rx) = mpsc::channel::<FsEvent>(64);

        let tick_tx = tx.clone();
        tokio::spawn(async move {
            let mut interval = tokio::time::interval(Duration::from_millis(500));
            loop {
                interval.tick().await;
                if tick_tx.send(AppEvent::Tick).await.is_err() { break; }
            }
        });

        let key_tx = tx.clone();
        tokio::task::spawn_blocking(move || {
            loop {
                if crossterm::event::poll(Duration::from_millis(100)).unwrap_or(false) {
                    if let Ok(CtEvent::Key(k)) = crossterm::event::read() {
                        if key_tx.blocking_send(AppEvent::Key(k)).is_err() { break; }
                    }
                }
            }
        });

        let fs_relay = tx.clone();
        tokio::spawn(async move {
            while let Some(ev) = fs_rx.recv().await {
                if fs_relay.send(AppEvent::Fs(ev)).await.is_err() { break; }
            }
        });

        Ok(Self { rx, fs_tx })
    }

    pub async fn next(&mut self) -> Option<AppEvent> {
        self.rx.recv().await
    }
}
```

```rust
//! Notify-based filesystem watcher for the TUI.
//!
//! SAFETY-INVARIANT-12: events are *signals to re-read*, never carry
//! file contents. The handlers always re-load through the storage API,
//! which holds an advisory lock and reads atomically.

use crate::event::FsEvent;
use notify::RecursiveMode;
use notify_debouncer_mini::{new_debouncer, DebounceEventResult, Debouncer};
use safessh_core::error::{Error, Result};
use safessh_storage::paths::Paths;
use std::time::Duration;
use tokio::sync::mpsc::Sender;

pub struct WatcherGuard(#[allow(dead_code)] Debouncer<notify::RecommendedWatcher>);

pub fn start_watcher(paths: &Paths, tx: Sender<FsEvent>) -> Result<WatcherGuard> {
    let pending_dir = paths.approvals_dir().join("pending");
    let projects_dir = paths.projects_dir();
    let audit_path = paths.audit_log();

    std::fs::create_dir_all(&pending_dir).ok();
    std::fs::create_dir_all(&projects_dir).ok();

    let mut debouncer = new_debouncer(Duration::from_millis(200), move |res: DebounceEventResult| {
        let Ok(events) = res else { return };
        let mut send = |ev: FsEvent| {
            let _ = tx.blocking_send(ev);
        };
        let mut saw_pending = false;
        let mut saw_projects = false;
        let mut saw_audit = false;
        for e in events {
            let p = &e.path;
            if p.starts_with(&pending_dir) { saw_pending = true; }
            else if p.starts_with(&projects_dir) { saw_projects = true; }
            else if p == &audit_path { saw_audit = true; }
        }
        if saw_pending { send(FsEvent::ApprovalsChanged); }
        if saw_projects { send(FsEvent::ProjectsChanged); }
        if saw_audit { send(FsEvent::AuditAppended); }
    }).map_err(|e| Error::Storage(format!("watcher: {e}")))?;

    debouncer.watcher().watch(&pending_dir, RecursiveMode::NonRecursive)
        .map_err(|e| Error::Storage(format!("watch pending: {e}")))?;
    debouncer.watcher().watch(&projects_dir, RecursiveMode::NonRecursive)
        .map_err(|e| Error::Storage(format!("watch projects: {e}")))?;
    // audit.log may not exist yet on first run; watch its parent dir.
    if let Some(parent) = audit_path.parent() {
        debouncer.watcher().watch(parent, RecursiveMode::NonRecursive)
            .map_err(|e| Error::Storage(format!("watch audit dir: {e}")))?;
    }

    Ok(WatcherGuard(debouncer))
}
```

```rust
use crate::theme;
use chrono::{Duration, Utc};
use ratatui::{prelude::*, widgets::*};
use safessh_core::error::{Error, Result};
use safessh_core::types::ApprovalToken;
use safessh_storage::approvals::*;
use safessh_storage::paths::Paths;

#[derive(Debug, Clone, Copy)]
pub enum PickerAction { Once, Timed(u32), Always, Deny, Block }

pub struct ApprovalsScreen {
    paths: Paths,
    pending: Vec<PendingRequest>,
    selected: usize,
    picker_open: bool,
    picker_idx: usize,
    pub timed_default_minutes: u32,
}

impl ApprovalsScreen {
    pub fn load(paths: &Paths) -> Result<Self> {
        let pending = list_pending(paths)?;
        Ok(Self {
            paths: Paths::user()?,
            pending, selected: 0,
            picker_open: false, picker_idx: 0,
            timed_default_minutes: 30,
        })
    }

    pub fn reload(&mut self) -> Result<()> {
        self.pending = list_pending(&self.paths)?;
        self.selected = self.selected.min(self.pending.len().saturating_sub(1));
        Ok(())
    }

    pub fn move_selection(&mut self, delta: i32) {
        if self.pending.is_empty() { return; }
        let len = self.pending.len() as i32;
        self.selected = (self.selected as i32 + delta).rem_euclid(len) as usize;
    }

    pub fn open_picker(&mut self) { if !self.pending.is_empty() { self.picker_open = true; self.picker_idx = 0; } }
    pub fn close_picker(&mut self) { self.picker_open = false; }
    pub fn picker_move(&mut self, delta: i32) { self.picker_idx = ((self.picker_idx as i32 + delta).rem_euclid(5)) as usize; }
    pub fn picker_action(&self) -> PickerAction {
        match self.picker_idx {
            0 => PickerAction::Once,
            1 => PickerAction::Timed(self.timed_default_minutes),
            2 => PickerAction::Always,
            3 => PickerAction::Deny,
            _ => PickerAction::Block,
        }
    }

    pub fn apply_to_selected(&mut self, action: PickerAction) -> Result<()> {
        let req = self.pending.get(self.selected)
            .ok_or_else(|| Error::Usage("nothing selected".into()))?
            .clone();
        // SAFETY-INVARIANT-12: every store write below holds the storage
        // crate's advisory lock and writes atomically; the TUI never bypasses
        // the API the CLI uses.
        let pending = PendingStore::new(&self.paths);
        let _ = pending.take(&ApprovalToken::from_str(&req.token));

        let pattern = PatternRule {
            rule_id: format!("rule-{}", Utc::now().timestamp_millis()),
            binary: req.parsed.binary.clone(),
            flags: req.parsed.flags.clone(),
            args_pattern: None,
            categories: req.categories.clone(),
            created_at: Utc::now(),
        };
        match action {
            PickerAction::Once => {}
            PickerAction::Timed(min) => {
                TimedStore::new(&self.paths).add(&req.project, TimedRule {
                    pattern, expires_at: Utc::now() + Duration::minutes(min as i64),
                })?;
            }
            PickerAction::Always => { AlwaysStore::new(&self.paths).add(&req.project, pattern)?; }
            PickerAction::Deny => { /* removing pending = deny-once */ }
            PickerAction::Block => { BlockedStore::new(&self.paths).add(&req.project, pattern)?; }
        }
        self.picker_open = false;
        self.reload()?;
        Ok(())
    }

    pub fn render(&self, frame: &mut Frame<'_>, area: Rect) {
        if self.pending.is_empty() {
            frame.render_widget(
                Paragraph::new("No pending approvals.").style(theme::dim()),
                area,
            );
            return;
        }
        let rows: Vec<Row> = self.pending.iter().map(|r| {
            let age = Utc::now() - r.created_at;
```

```rust
use safessh_core::error::{Error, Result};
use safessh_storage::paths::Paths;
use serde_json::Value;
use std::fs::File;
use std::io::{BufRead, BufReader, Seek, SeekFrom};

const TAIL_LIMIT: usize = 200;

#[derive(Clone, Debug)]
pub struct AuditRow {
    pub ts: chrono::DateTime<chrono::Utc>,
    pub event_type: String,
    pub project: String,
    pub summary: String,
    pub raw_json: String,
}

#[derive(Default, Clone, Debug)]
pub struct Filters {
    pub project: Option<String>,
    pub event_type: Option<String>,
    pub grep: Option<String>,
}

pub struct AuditScreen {
    paths: Paths,
    rows: Vec<AuditRow>,
    offset_bytes: u64,
    selected: usize,
    auto_scroll: bool,
    pub filters: Filters,
    pub editing: Option<EditField>,
    pub edit_buf: String,
}

#[derive(Copy, Clone, Debug)]
pub enum EditField { Project, Type, Grep }

impl AuditScreen {
    pub fn load(paths: &Paths) -> Result<Self> {
        let mut s = Self {
            paths: Paths::user()?,
            rows: vec![], offset_bytes: 0,
            selected: 0, auto_scroll: true,
            filters: Filters::default(), editing: None, edit_buf: String::new(),
        };
        s.full_reload()?;
        Ok(s)
    }

    fn full_reload(&mut self) -> Result<()> {
        let path = self.paths.audit_log();
        if !path.exists() { return Ok(()); }
        let file = File::open(&path).map_err(Error::Io)?;
        let reader = BufReader::new(file);
        let mut all: Vec<AuditRow> = Vec::new();
        let mut total_bytes: u64 = 0;
        for line in reader.lines() {
            let Ok(line) = line else { continue };
            total_bytes += line.len() as u64 + 1;
            if let Some(row) = parse_row(&line) { all.push(row); }
        }
        let start = all.len().saturating_sub(TAIL_LIMIT);
        self.rows = all.split_off(start);
        self.offset_bytes = total_bytes;
        self.selected = self.rows.len().saturating_sub(1);
        Ok(())
    }

    pub fn append_tail(&mut self) -> Result<()> {
        let path = self.paths.audit_log();
        if !path.exists() { return Ok(()); }
        let mut file = File::open(&path).map_err(Error::Io)?;
        let new_len = file.metadata().map_err(Error::Io)?.len();
        if new_len < self.offset_bytes {
            // Rotated — start fresh.
            return self.full_reload();
        }
        file.seek(SeekFrom::Start(self.offset_bytes)).map_err(Error::Io)?;
        let reader = BufReader::new(file);
        for line in reader.lines() {
            let Ok(line) = line else { continue };
            self.offset_bytes += line.len() as u64 + 1;
            if let Some(row) = parse_row(&line) {
                self.rows.push(row);
                if self.rows.len() > TAIL_LIMIT { self.rows.remove(0); }
            }
        }
        if self.auto_scroll { self.selected = self.rows.len().saturating_sub(1); }
        Ok(())
    }

    pub fn move_selection(&mut self, delta: i32) {
        let len = self.rows.len() as i32;
        if len == 0 { return; }
        let next = (self.selected as i32 + delta).clamp(0, len - 1);
        self.selected = next as usize;
        self.auto_scroll = self.selected as i32 == len - 1;
    }

    pub fn jump_top(&mut self) { self.selected = 0; self.auto_scroll = false; }
    pub fn jump_bottom(&mut self) { self.selected = self.rows.len().saturating_sub(1); self.auto_scroll = true; }

    pub fn start_edit(&mut self, field: EditField) {
        self.editing = Some(field);
        self.edit_buf.clear();
    }

    pub fn commit_edit(&mut self) {
        let val = if self.edit_buf.is_empty() { None } else { Some(self.edit_buf.clone()) };
        match self.editing {
            Some(EditField::Project) => self.filters.project = val,
            Some(EditField::Type) => self.filters.event_type = val,
            Some(EditField::Grep) => self.filters.grep = val,
            None => {}
        }
        self.editing = None;
        self.edit_buf.clear();
    }

    fn visible(&self) -> Vec<&AuditRow> {
        self.rows.iter().filter(|r| {
            self.filters.project.as_deref().map_or(true, |p| r.project == p)
            && self.filters.event_type.as_deref().map_or(true, |t| r.event_type == t)
            && self.filters.grep.as_deref().map_or(true, |g| r.raw_json.contains(g))
        }).collect()
    }

    pub fn render(&self, frame: &mut Frame<'_>, area: Rect) {
        let v = self.visible();
        if v.is_empty() && self.rows.is_empty() {
            frame.render_widget(Paragraph::new("No audit events yet.").style(theme::dim()), area);
            return;
        }
        let chunks = Layout::vertical([Constraint::Length(1), Constraint::Min(0), Constraint::Length(1)]).split(area);

        let f = format!(
            "filters:  project={}  type={}  grep={}",
            self.filters.project.as_deref().unwrap_or("-"),
            self.filters.event_type.as_deref().unwrap_or("-"),
            self.filters.grep.as_deref().unwrap_or("-"),
        );
        frame.render_widget(Paragraph::new(f).style(theme::dim()), chunks[0]);

        let rows: Vec<Row> = v.iter().map(|r| Row::new(vec![
            r.ts.format("%H:%M:%S").to_string(),
            r.event_type.clone(),
            r.project.clone(),
            r.summary.clone(),
        ])).collect();
        let widths = [Constraint::Length(8), Constraint::Length(20), Constraint::Length(14), Constraint::Min(20)];
        let mut state = TableState::default();
        if !v.is_empty() { state.select(Some(self.selected.min(v.len() - 1))); }
        let table = Table::new(rows, widths)
            .header(Row::new(vec!["TIME", "EVENT", "PROJECT", "SUMMARY"]).style(theme::title()))
            .block(Block::default().borders(Borders::ALL))
            .row_highlight_style(theme::title());
        frame.render_stateful_widget(table, chunks[1], &mut state);

        let bottom = if let Some(field) = self.editing {
            let label = match field { EditField::Project => "project", EditField::Type => "type", EditField::Grep => "grep" };
            format!("/{label}: {}_   [Enter apply, Esc cancel]", self.edit_buf)
        } else {
            "/p project  /t type  /g grep   gG top/bot  d↑↓".into()
        };
        frame.render_widget(Paragraph::new(bottom).style(theme::dim()), chunks[2]);
    }
}

fn parse_row(line: &str) -> Option<AuditRow> {
    let v: Value = serde_json::from_str(line).ok()?;
    let event_type = v.get("event_type")?.as_str()?.to_string();
    let project = v.get("project").and_then(|p| p.as_str()).unwrap_or("-").to_string();
    let ts = v.get("ts").and_then(|t| t.as_str())
        .and_then(|s| chrono::DateTime::parse_from_rfc3339(s).ok())
        .map(|t| t.with_timezone(&chrono::Utc))
        .unwrap_or_else(|| chrono::Utc::now());
    let summary = match event_type.as_str() {
        "exec_attempt" => v.pointer("/data/binary").and_then(|x| x.as_str()).unwrap_or("").to_string(),
        "exec_complete" => format!("exit={} dur={}ms",
            v.pointer("/data/exit_code").and_then(|x| x.as_i64()).unwrap_or(0),
            v.pointer("/data/duration_ms").and_then(|x| x.as_u64()).unwrap_or(0)),
        "approval_requested" => format!("token={}",
            v.pointer("/data/token").and_then(|x| x.as_str()).unwrap_or("")),
        "yolo_invocation" => v.pointer("/data/raw").and_then(|x| x.as_str()).unwrap_or("").to_string(),
        _ => String::new(),
    };
    Some(AuditRow { ts, event_type, project, summary, raw_json: line.to_string() })
}
```

```rust
pub struct ImportDialog {
    pub aliases: Vec<safessh_storage::ssh_config::SshAlias>,
    pub checked: Vec<bool>,
    pub cursor: usize,
}

impl ProjectsScreen {
    pub fn open_import(&mut self) -> Result<()> {
        let snap = safessh_storage::ssh_config::SshConfigSnapshot::load(&self.paths)?;
        let len = snap.aliases.len();
        self.import = Some(ImportDialog { aliases: snap.aliases, checked: vec![false; len], cursor: 0 });
        Ok(())
    }
    pub fn close_import(&mut self) { self.import = None; }
    pub fn import_move(&mut self, delta: i32) {
        if let Some(d) = self.import.as_mut() {
            if d.aliases.is_empty() { return; }
            d.cursor = ((d.cursor as i32 + delta).rem_euclid(d.aliases.len() as i32)) as usize;
        }
    }
    pub fn import_toggle(&mut self) {
        if let Some(d) = self.import.as_mut() {
            if let Some(c) = d.checked.get_mut(d.cursor) { *c = !*c; }
        }
    }
    pub fn import_commit(&mut self) -> Result<usize> {
        let dialog = match self.import.take() { Some(d) => d, None => return Ok(0) };
        let store = ProjectStore::new(Paths::user()?);
        let existing: std::collections::HashSet<String> = store.list().unwrap_or_default().into_iter().collect();
        let mut created = 0;
        for (i, alias) in dialog.aliases.iter().enumerate() {
            if !dialog.checked[i] { continue; }
            if existing.contains(&alias.alias) { continue; }
            let target = Target::Inline {
                name: "default".into(),
                host: alias.hostname.clone().unwrap_or_else(|| alias.alias.clone()),
                port: alias.port.unwrap_or(22),
                user: alias.user.clone().unwrap_or_else(|| std::env::var("USER").unwrap_or_default()),
                identity_file: alias.identity_file.clone(),
                proxy_jump: None,        // ssh2-config 0.3 doesn't expose ProxyJump; see SshAlias doc.
                keychain_secret: None,
            };
            let project = Project {
                name: alias.alias.clone(),
                default_target: "default".into(),
                targets: vec![target],
                policy: Policy {
                    allow: vec!["read:safe".into(), "file:read".into()],
                    require_approval: vec![], deny: vec![],
                },
                approvals: Approvals::default(),
                output: OutputCaps::default(),
            };
            store.save(&project)?;
            created += 1;
        }
        self.reload()?;
        Ok(created)
    }
}
```

```rust
    pub fn show_toast(&mut self, text: impl Into<String>) {
        self.toast = Some(Toast {
            text: text.into(),
            expires_at: chrono::Utc::now() + chrono::Duration::milliseconds(3000),
```
